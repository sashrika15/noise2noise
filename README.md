# noise2noise

This is a PyTorch implementation of noise2noise. 

Paper: [noise2noise](https://arxiv.org/abs/1803.04189).

### Test
To run test on images in /test_images, run command:
```
python n2n_test.py
```
The noise type can be edited in n2n_test.py
